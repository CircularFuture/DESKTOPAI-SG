# DIRTREE_depth4_plus

Directory-only listing for paths deeper than depth 3, grouped by top-level folder. Excludes build/cache/VCS and binary/media folders to keep file size manageable.

Excluded directories: `node_modules`, `dist`, `build`, `out`, `.next`, `coverage`, `.git`, `.turbo`, `.cache`, `venv`, and common media/binary folders.
